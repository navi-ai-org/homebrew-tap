# navi-ai-org Homebrew Tap

Homebrew formulae for **[NAVI](https://github.com/navi-ai-org/navi)** — a coding agent engine with a terminal UI.

## Install

```bash
brew install navi-ai-org/tap/navi
```

Or add the tap first:

```bash
brew tap navi-ai-org/tap
brew install navi
```

> **Note:** Homebrew core also ships a different formula named `navi` (an interactive cheatsheet). Prefer the fully qualified form `navi-ai-org/tap/navi` to get this project.

## Upgrade

```bash
brew update
brew upgrade navi-ai-org/tap/navi
```
